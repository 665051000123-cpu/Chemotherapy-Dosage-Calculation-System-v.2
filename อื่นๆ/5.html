1. User Interface

หน้าเว็บระบบ & Dashboard

React SPA / Print Agent

แพทย์/เภสัชกร ดูสถิติ คีย์สั่งยา และตั้งค่าเครื่องพิมพ์
2. Route

กำหนดเส้นทาง

Express Endpoint (server.js)

ชี้เป้าหมาย URL ประวัติสั่งยา, สถิติ, และการพิมพ์
3. Middleware
JWT Authentication
Admin/User Verification
Express JSON parser
CORS Security
ตรวจสอบสิทธิ์ Admin ในการดึง Audit Log
4. Controller

รับ Request

ถอด req.body / req.params

รับพารามิเตอร์การคำนวณ, ตัวทำละลาย, ID เครื่องพิมพ์
5. Service Layer
คำนวณ BSA & Dose ยา
ตรวจสอบ Overdose
จัดการคิวพิมพ์ Offline Agent
ประมวลผลสถิติแดชบอร์ด
คำนวณสูตรร่วมกับสารละลายและสอบทานความปลอดภัย
6. Model

Data Mapper

จัดโครงสร้าง SQL Query Object

เตรียมคำสั่งเขียน/อ่าน และดึงประวัติรอบยา (Cycles)
7. Database

MySQL DB

patients, drugs, order_logs, solvents, default_printer, audit_logs
